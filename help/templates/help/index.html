Organisations

Organisations are any group that wishes to make their events or resources available to be searched. An Organisation must have a designated admin who will be responsible for maintaining information regarding the Organisation.

An admin may manage multiple organisations, it is not recommended and may be removed in a future version of the application, but as of right now, an admin can manage multiple organisations. A user may not create organisations or take control of organisations, an application developer must manually assign organisations to admins.

Dashboard

The dashboard is the principle means by which admins will manage their Organisation(s). Once an admin has logged in, the application will automatically send them to the dashboard which will list the Organisation(s) they manage, typically each admin will only have one Organisation, but at this time it is possible an admin may manage multiple Organisations.
